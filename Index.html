About Me and My Work

I am currently working as a Technical Service Analyst for Superior Digital Solutions as well as a part time Service Technician for Xerox. My work includes the setup of Xerox printers at various client sites as well as in-depth troubleshooting. Troubleshooting encompasses image quality issues, electrical as well as mechanical problems, and even as simple as paper jams. In my previous work as a developer, I designed and developed a project management web application for Essar Steel. I moved the existing Access back-end over to an MS SQL server back-end, and also created an ASP.NET MVC front-end. I designed and developed extensively using C#, Javascript, jQuery, AJAX and HTML, while using an agile approach.

I also enjoy programming and find spare time to make my own programs, or work on my current video game project. My programming background involves Java, which I used throughout most of the courses it took while completing a Bachelor of Computer Science at Algoma University. Additionally, as a final year project I worked in a group to create a video game using the Unity3d game engine, which uses C# for scripting. My current projects include a video game and an editing tool for adjusting meta-data for music files.
